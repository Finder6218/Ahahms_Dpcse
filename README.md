# 信访管理系统--区人大专用版

管理内容包括：

1.来访管理（来访管理包括：来访登记，按来访分类查询，期间来访汇总）

2.来信管理（来信管理包括：来信登记，按来信类别查询，期间来信汇总）。

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
